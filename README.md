# UTBK Streamlit Portfolio (Final Version)

Aplikasi portofolio interaktif dengan Streamlit yang menampilkan analisis data UTBK, pelatihan model prediksi kelulusan, dan visualisasi interaktif.

## Fitur
- Halaman **About** & **Projects**
- Halaman **Data Viz** untuk eksplorasi dataset
- Halaman **Train Model** untuk melatih Logistic Regression otomatis
- Halaman **Predict** untuk mengunggah data baru dan melakukan prediksi

## Cara Menjalankan
```bash
pip install -r requirements.txt
streamlit run app.py
```
